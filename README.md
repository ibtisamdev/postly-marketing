# Postly Marketing

Demo marketing site for a social media scheduling platform, built with Next.js and Sanity.

## Tech Stack

- Next.js 16 (App Router, React 19, Turbopack)
- Sanity v4 (next-sanity v11)
- Tailwind CSS 4
- Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
